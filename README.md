# Vacuagenda

A simple way to keep track of shots taken during a medical treatment.

https://joachim92.github.io/vacuagenda/

All the data is stored locally in the web browser.

Can be added as a website app on iphone following these [instructions](https://support.apple.com/guide/iphone/open-as-web-app-iphea86e5236/26/ios/26).
